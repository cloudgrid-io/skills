# Security policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, report it
responsibly. Do not open a public issue.

Email **security@cloudgrid.io** with a description of the issue, steps to
reproduce, and any relevant logs or screenshots. You will receive an
acknowledgement within 48 hours and a detailed response within 5 business
days.

## Scope

This policy covers the code and configuration in this repository. The
CloudGrid platform (API, runtime, infrastructure) has its own security
process — report platform issues to the same address.

## Supported versions

Only the latest release on `main` is supported. Apply updates promptly.

## Disclosure

We follow coordinated disclosure. We will work with you on a timeline and
credit you in the advisory unless you prefer otherwise.
